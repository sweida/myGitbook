Introduction

全局安装

npm install gitbook-cli -g

初始化

gitbook init

修改端口号并启动

gitbook serve --port 8000

使用

目录添加数字

{
  "pluginsConfig": {
    "theme-default": {
      "showLevel": true
    }
  }
}

book.json配置

{
  "title": "我的第一本电子书哈哈",
  "description": "用 GitBook 制作的第一本電子書！",
  "isbn": "978-3-16-148410-0",
  "language": "zh",
  "direction": "ltr",
  "pluginsConfig": {
    "theme-default": {
      "showLevel": true
    }
  }
}

打包

gitbook build
